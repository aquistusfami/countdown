# ⏳ Đếm Ngược THPT Quốc Gia 2026

Trang web đếm ngược thời gian còn lại đến kỳ thi Tốt nghiệp THPT Quốc Gia 2026.

## 🚀 Cách dùng

Mở file `index.html` trực tiếp trên trình duyệt — không cần cài đặt hay server.

```
THPT_QG/
└── index.html   # Toàn bộ trang web (HTML + CSS + JS)
```

## ✨ Tính năng

- Đếm ngược real-time đến **7:30 ngày 11/6/2026**
- Thanh tiến độ ôn thi (tính từ 1/1/2026)
- Thông tin kỳ thi: ngày thi, công bố kết quả, môn thi
- Câu quote động lực tự động thay sau mỗi 8 giây
- Hiệu ứng nền sao chuyển động, giao diện dark mode

## 📅 Lịch thi THPT QG 2026

| Sự kiện | Ngày |
|---|---|
| Ngày thi | 11 – 12/6/2026 |
